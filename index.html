GPS Safety Watch

The goal of this Tool is to create a "safety device for a person with GPS tracking location" that will allow police to track offenders using the safety device with GPS tracking by sending real-time SMS alerts and GPS tracking to the nearest police station. Depending on the user's response, the device can prevent crimes before they happen. The gps tracking system can be used for safety when the wearer is in danger it can send a message that he is in danger
